## ADDED Requirements

### Requirement: Asset gallery empty state shall be visually centered

When the asset gallery has no assets matching the current filter, the empty state
message MUST be visually centered within the gallery container.

#### Scenario: Empty gallery shows centered placeholder text

- **WHEN** the asset gallery has no assets
- **THEN** an empty state message is displayed
- **AND** the message is horizontally and vertically centered within the gallery container
